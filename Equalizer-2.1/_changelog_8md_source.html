204 # Release 1.8 (14-Oct-2014)
205 
206 * Implemented Qt window system for onscreen windows. Issue
207  [21](https://github.com/Eyescale/Equalizer/issues/21) partially done.
208 * DisplayCluster streaming can be enabled with automatic configuration using new
209  global view attributes: EQ_VIEW_SATTR_DISPLAYCLUSTER and
210  EQ_VIEW_SATTR_PIXELSTREAM_NAME.

212  downstream projects
213 * Compression is enabled for DisplayCluster streaming
214 * DisplayCluster streaming is now asynchronous
