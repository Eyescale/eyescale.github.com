53 # Release 1.0 (24-Jun-2013)
54 
55 * 07/Mar/2013: PluginRegistry, Plugin and compressors are moved to
56  Lunchbox. co::Global still maintains the global Collage plugin
57  registry.
58 
59 * 11/Feb/2013: Node::useMulticast has been renamed to getMulticast. This
60  method, and the previous getMulticast have been made protected.
61  Node::getConnection has a flag to prefer a multicast connection.
62 
63 * 28/Jan/2013: The program name and working directory have been moved
64  from co::Global to eq::Global.
65 
66 * 06/Sep/2012: New stream-based commands supersedes packet-based
67  messaging. New send() methods in co::Node & co::Object replaces old
68  API. All packets are superseded by NodeOCommand & ObjectOOCommand for
69  sending commands, and by Command & ObjectCommand for receiving
70  commands.
71 
72 * 09/Aug/2012: Made co::DataOStream::write private. Use 'os <<
73  co::Array< T >( ptr, num )' instead.
74 
75 * 27/Jul/2012: Made co::DataIStream::read private. Use 'is >> co::Array<
76  T >( ptr, num )' instead.
