6 # Introduction {#Introduction}
7 
8 Welcome to Equalizer, the standard middleware to create and deploy
9 parallel, scalable OpenGL applications. This release introduces major
10 new features, most notably integrated head tracking using VRPN or OpenCV.
11 
12 Equalizer 1.7 is a feature release extending the 1.0 API, distilling
13 over several years of development and decades of experience into a
14 feature-rich, high-performance and mature parallel rendering
15 framework. It is intended for all application developers creating
16 parallel, interactive OpenGL applications. Equalizer 1.7 can be
17 retrieved by downloading the [source code](http://www.equalizergraphics.com/downloads/Equalizer-1.7.0.tar.gz") or one of the [precompiled packages](http://www.equalizergraphics.com/downloads/major.html).
18 
19 ## Features {#Features}
20 
21 Equalizer provides the following major features to facilitate the development and deployment of scalable OpenGL applications. A [detailed feature list](http://www.equalizergraphics.com/features.html) can be found on the [Equalizer website](http://www.equalizergraphics.com).
22 
23 * Runtime Configurability: An Equalizer application is configured
24  automatically or manually at runtime and can be deployed on laptops,
25  multi-GPU workstations and large-scale visualization clusters without
26  recompilation.
27 * Runtime Scalability: An Equalizer application can benefit from
28  multiple graphics cards, processors and computers to scale rendering
29  performance, visual quality and display size.
30 * Distributed Execution: Equalizer applications can be written to
31  support cluster-based execution. Equalizer uses the
32  [Collage network library](http://www.libcollage.net), a cross-platform
33  C++ library for building heterogenous, distributed applications.
34 * Support for Stereo and Immersive Environments: Equalizer supports
35  stereo rendering head tracking, head-mounted displays and other
36  advanced features for immersive Virtual Reality installations.

118 # About {#About}
119 
120 Equalizer is a cross-platform framework, designed to run on any modern
121 operating system, including all Unix variants and the Windows operating
122 system. Equalizer requires at least [OpenGL 1.1](http://www.opengl.org),
123 but uses newer OpenGL features when available. Equalizer uses CMake and
124 [Buildyard](https://github.com/Eyescale/Buildyard) to create a
125 platform-specific build environment. The following platforms and build
126 environments are tested for version 1.7:
127 
128 * Linux: Ubuntu 13.10, 14.04, RHEL 6.5 (Makefile, i386, x64)
129 * Windows: 7 (Visual Studio 2008, i386, x64)
130 * Mac OS X: 10.8 (Makefile, XCode, i386, x64)
131 
132 The Equalizer Programming and User Guide covers the basics of Collage
133 programming. The API documentation can be found on
134 [github](http://eyescale.github.com/).
135 
136 As with any open source project, the available source code, in
137 particular the shipped examples provide a reference for developing or
138 porting applications.
139 
140 Technical questions can be posted to the eq-dev Mailing List, or
141 directly to info@equalizergraphics.com.
142 
143 Commercial support, custom software development and porting services are
144 available from [Eyescale](http://www.eyescale.ch). Please contact
145 [info@eyescale.ch](mailto:info@eyescale.ch?subject=Collage%20support)
146 for further information.
