75 # Release 1.8 (14-10-2014) {#Release18}
76 
77 * Implemented Qt window system for onscreen windows. Issue
78  [21](https://github.com/Eyescale/Equalizer/issues/21) partially done.
79 * DisplayCluster streaming can be enabled with automatic configuration using new
80  global view attributes: EQ_VIEW_SATTR_DISPLAYCLUSTER and
81  EQ_VIEW_SATTR_PIXELSTREAM_NAME.
82 * New EqGLLibraries.cmake script for simpler OpenGL configuration in
83  downstream projects
84 * Compression is enabled for DisplayCluster streaming
85 * DisplayCluster streaming is now asynchronous
