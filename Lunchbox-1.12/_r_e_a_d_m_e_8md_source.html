3 # Introduction {#Introduction}
4 
5 Welcome to Lunchbox, a C++ library for multi-threaded programming. Lunchbox was
6 formerly known as eq::base or co::base, the foundation for the
7 [Equalizer parallel rendering framework](https://github.com/Eyescale/Equalizer)
8 and the [Collage network library](https://github.com/Eyescale/Collage). It is
9 intended for all application developers creating high-performance multi-threaded
10 programs.
11 
12 The
13 [API documentation](http://eyescale.github.io/Lunchbox-1.11/index.html) can be
14 found on [eyescale.github.io](http://eyescale.github.io/). As with any open
15 source project, the available source code, in particular the
16 [unit tests](https://github.com/Eyescale/Lunchbox/tree/1.11/tests) provide a
17 reference for developing applications.
18 
19 Technical questions can be posted to the Equalizer
20 [Developer Mailing List](http://www.equalizergraphics.com/cgi-bin/mailman/listinfo/eq-dev),
21 or directly to
22 [info@equalizergraphics.com](mailto:info@equalizergraphics.com?subject=Lunchbox%20question).
23 
24 Commercial support, custom software development and porting services are
25 available from [Eyescale](http://www.eyescale.ch). Please contact
26 [info@eyescale.ch](mailto:info@eyescale.ch?subject=Lunchbox%20support)
27 for further information.

29 # Features {#Features}
30 
31 Lunchbox provides the following major features to facilitate the
32 development and deployment of multi-threaded applications:
33 * Operating System Abstraction: lunchbox::Atomic, lunchbox::Condition,
34  lunchbox::DSO, @ref bitops "bit operations", lunchbox::daemonize(),
35  (lunchbox::Clock, lunchbox::MemoryMap, lunchbox::PerThread, lunchbox::RNG,
36  lunchbox::Thread)
37 * High-Performance Threading Primitives: lunchbox::Buffer, lunchbox::LFQueue,
38  lunchbox::LFVector, lunchbox::Monitor, lunchbox::MTQueue,
39  lunchbox::RequestHandler, lunchbox::SpinLock, (lunchbox::Lock,
40  lunchbox::TimedLock)
41 * Utility classes: lunchbox::Any, lunchbox::Log, lunchbox::Pool,
42  lunchbox::UnorderedIntervalSet, lunchbox::Future, lunchbox::PersistentMap,
43  (lunchbox::ScopedMutex)
44 
45 # Building {#Building}
46 
47 Lunchbox is a cross-platform library, designed to run on any modern operating
48 system, including all Unix variants and the Windows operating system. Lunchbox
49 uses CMake to create a platform-specific build environment. The following
50 platforms and build environments are tested:
51 
52 * Linux: Ubuntu 14.04, RHEL 6.6 (Makefile, Ninja)
53 * Windows: 7 (Visual Studio 2012)
54 * Mac OS X: 10.8 (Makefile, Ninja)
55 
56 ~~~
57 git clone https://github.com/Eyescale/Lunchbox.git
58 cd Lunchbox
59 mkdir build
60 cd build
61 cmake ..
62 make
63 ~~~
