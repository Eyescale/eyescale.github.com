104 # Release 1.8 (14-10-2014) {#Release18}
105 
106 * Implemented Qt window system for onscreen windows. Issue
107  [21](https://github.com/Eyescale/Equalizer/issues/21) partially done.
108 * DisplayCluster streaming can be enabled with automatic configuration using new
109  global view attributes: EQ_VIEW_SATTR_DISPLAYCLUSTER and
110  EQ_VIEW_SATTR_PIXELSTREAM_NAME.
111 * New EqGLLibraries.cmake script for simpler OpenGL configuration in
112  downstream projects
113 * Compression is enabled for DisplayCluster streaming
114 * DisplayCluster streaming is now asynchronous
