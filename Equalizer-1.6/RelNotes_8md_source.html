6 # Introduction {#Introduction}
7 
8 Welcome to Equalizer, the standard middleware to create and deploy
9 parallel, scalable OpenGL applications. This release introduces major
10 new features, most notably asynchronous readbacks, region of interest
11 and thread affinity for increased performance during scalable
12 rendering.
13 
14 Equalizer 1.6 is a feature release extending the 1.0 API, distilling
15 over eight years of development and decades of experience into a
16 feature-rich, high-performance and mature parallel rendering
17 framework. It is intended for all application developers creating
18 parallel, interactive OpenGL applications. Equalizer 1.6 can be
19 retrieved by downloading the [source code](http://www.equalizergraphics.com/downloads/Equalizer-1.6.0.tar.gz") or one of the [precompiled packages](http://www.equalizergraphics.com/downloads/major.html).
20 
21 ## Features {#Features}
22 
23 Equalizer provides the following major features to facilitate the development and deployment of scalable OpenGL applications. A [detailed feature list](http://www.equalizergraphics.com/features.html) can be found on the [Equalizer website](http://www.equalizergraphics.com).
24 
25 * Runtime Configurability: An Equalizer application is configured
26  automatically or manually at runtime and can be deployed on laptops,
27  multi-GPU workstations and large-scale visualization clusters without
28  recompilation.
29 * Runtime Scalability: An Equalizer application can benefit from
30  multiple graphics cards, processors and computers to scale rendering
31  performance, visual quality and display size.
32 * Distributed Execution: Equalizer applications can be written to
33  support cluster-based execution. Equalizer uses the
34  [Collage network library](http://www.libcollage.net), a cross-platform
35  C++ library for building heterogenous, distributed applications.
36 * Support for Stereo and Immersive Environments: Equalizer supports
37  stereo rendering head tracking, head-mounted displays and other
38  advanced features for immersive Virtual Reality installations.

132 # About {#About}
133 
134 Equalizer is a cross-platform framework, designed to run on any modern
135 operating system, including all Unix variants and the Windows operating
136 system. Equalizer requires at least [OpenGL 1.1](http://www.opengl.org),
137 but uses newer OpenGL features when available. Equalizer uses CMake and
138 [Buildyard](https://github.com/Eyescale/Buildyard) to create a
139 platform-specific build environment. The following platforms and build
140 environments are tested for version 1.6:
141 
142 * Linux: Ubuntu 12.04, 12.10, RHEL 6.4 (Makefile, i386, x64)
143 * Windows: 7 (Visual Studio 2008, i386, x64)
144 * Mac OS X: 10.8 (Makefile, XCode, i386, x64)
145 
146 The Equalizer Programming and User Guide covers the basics of Collage
147 programming. The API documentation can be found on
148 [github](http://eyescale.github.com/).
149 
150 As with any open source project, the available source code, in
151 particular the shipped examples provide a reference for developing or
152 porting applications.
153 
154 Technical questions can be posted to the eq-dev Mailing List, or
155 directly to info@equalizergraphics.com.
156 
157 Commercial support, custom software development and porting services are
158 available from [Eyescale](http://www.eyescale.ch). Please contact
159 [info@eyescale.ch](mailto:info@eyescale.ch?subject=Collage%20support)
160 for further information.
