6 # Introduction {#Introduction}
7 
8 Welcome to Equalizer, the standard middleware to create and deploy
9 parallel, scalable OpenGL applications. This release introduces major
10 new features, most notably integrated head tracking using VRPN or OpenCV.
11 
12 Equalizer 1.6 is a feature release extending the 1.0 API, distilling
13 over eight years of development and decades of experience into a
14 feature-rich, high-performance and mature parallel rendering
15 framework. It is intended for all application developers creating
16 parallel, interactive OpenGL applications. Equalizer 1.6 can be
17 retrieved by downloading the [source code](http://www.equalizergraphics.com/downloads/Equalizer-1.6.0.tar.gz") or one of the [precompiled packages](http://www.equalizergraphics.com/downloads/major.html).
18 
19 ## Features {#Features}
20 
21 Equalizer provides the following major features to facilitate the development and deployment of scalable OpenGL applications. A [detailed feature list](http://www.equalizergraphics.com/features.html) can be found on the [Equalizer website](http://www.equalizergraphics.com).
22 
23 * Runtime Configurability: An Equalizer application is configured
24  automatically or manually at runtime and can be deployed on laptops,
25  multi-GPU workstations and large-scale visualization clusters without
26  recompilation.
27 * Runtime Scalability: An Equalizer application can benefit from
28  multiple graphics cards, processors and computers to scale rendering
29  performance, visual quality and display size.
30 * Distributed Execution: Equalizer applications can be written to
31  support cluster-based execution. Equalizer uses the
32  [Collage network library](http://www.libcollage.net), a cross-platform
33  C++ library for building heterogenous, distributed applications.
34 * Support for Stereo and Immersive Environments: Equalizer supports
35  stereo rendering head tracking, head-mounted displays and other
36  advanced features for immersive Virtual Reality installations.

130 # About {#About}
131 
132 Equalizer is a cross-platform framework, designed to run on any modern
133 operating system, including all Unix variants and the Windows operating
134 system. Equalizer requires at least [OpenGL 1.1](http://www.opengl.org),
135 but uses newer OpenGL features when available. Equalizer uses CMake and
136 [Buildyard](https://github.com/Eyescale/Buildyard) to create a
137 platform-specific build environment. The following platforms and build
138 environments are tested for version 1.6:
139 
140 * Linux: Ubuntu 12.04, 12.10, RHEL 6.4 (Makefile, i386, x64)
141 * Windows: 7 (Visual Studio 2008, i386, x64)
142 * Mac OS X: 10.8 (Makefile, XCode, i386, x64)
143 
144 The Equalizer Programming and User Guide covers the basics of Collage
145 programming. The API documentation can be found on
146 [github](http://eyescale.github.com/).
147 
148 As with any open source project, the available source code, in
149 particular the shipped examples provide a reference for developing or
150 porting applications.
151 
152 Technical questions can be posted to the eq-dev Mailing List, or
153 directly to info@equalizergraphics.com.
154 
155 Commercial support, custom software development and porting services are
156 available from [Eyescale](http://www.eyescale.ch). Please contact
157 [info@eyescale.ch](mailto:info@eyescale.ch?subject=Collage%20support)
158 for further information.
