6 # Introduction {#Introduction}
7 
8 Welcome to Lunchbox, a C++ library for multi-threaded programming.
9 Lunchbox 1.8 adds major new features, most notably runtime-loadable
10 compression plugins and an UnorderedIntervalSet container.
11 
12 Lunchbox 1.8 is a feature release based on the 1.6 Lunchbox API. It is
13 intended for all application developers creating high-preformance
14 multi-threaded programs. Lunchbox 1.8 can be retrieved by downloading
15 the [source
16 code](http://www.equalizergraphics.com/downloads/Lunchbox-1.8.0.tar.gz)
17 [precompiled
18 packages](http://www.equalizergraphics.com/downloads/major.html).
19 
20 ## Features {#Features}
21 
22 Lunchbox provides the following major features to facilitate the
23 development and deployment of multi-threaded applications:
24 
25 * OS Abstraction: Utility classes abstracting common operating systems
26  features, e.g., threads, locks, memory maps, shared library loading
27  and condition variables.
28 * High-Performance Primitives: Thread-safe utilities tuned for
29  performance, e.g., atomic variables, spin locks and lock-free
30  containers.
31 * Utility classes: Helper primitives which are not in the standard
32  library, e.g., logging, pools and random number generation.

83 # About {#About}
84 
85 Lunchbox is a cross-platform library, designed to run on any modern
86 operating system, including all Unix variants and the Windows operating
87 system. Lunchbox uses CMake to create a platform-specific build
88 environment. The following platforms and build environments are tested:
89 
90 * Linux: Ubuntu 12.04, 12.10, 13.04, RHEL 6 (Makefile, i386, x64)
91 * Windows: 7 (Visual Studio 2008, i386, x64)
92 * Mac OS X: 10.8 (Makefile, XCode, i386, x64)
93 
94 The
95 [API documentation](http://eyescale.github.com/Lunchbox-1.8/index.html)
96 can be found on [eyescale.github.com](http://eyescale.github.com/). As
97 with any open source project, the available source code, in particular
98 the [unit tests](https://github.com/Eyescale/Lunchbox/tree/1.8/tests)
99 provide a reference for developing applications.
